PAP Oficina 3P - KitsunaTalesOfSpirits Jogo 2D Unity
---------------------------------------------------------------------------------------------------------------------------------------------------------

Este projeto está a ser desenvolvido no âmbito da Prova de Aptidão Profissional (PAP), 
consiste em um jogo 2D do género Metroidvania, desenvolvido em Unity com a linguagem de programação C#, 
inspirado na mitologia japonesa.

O jogo baseia-se numa progressão não linear, incentivando a exploração do mapa e a descoberta de caminhos alternativos. 
À medida que o jogador avança, desbloqueia habilidades que permitem o acesso a novas áreas, sendo assim o jogo é dividido em diferentes biomas, 
cada um com características próprias, inimigos específicos e desafios distintos.

Ao longo da aventura, o jogador poderá encontrar diversos itens, abrir baús, resolver puzzles e enfrentar inimigos e bosses. 
O sistema de itens é gerido através de ScriptableObjects (ItemSO), permitindo uma organização eficiente dos dados do jogo. 

A componente narrativa é inspirada em elementos espirituais e mitológicos do folclore japonês, sendo complementada por cutscenes.
A parte visual do jogo é desenvolvida com recurso ao LibreSprite, utilizado na criação de sprites e animações em pixel art.

Para garantir uma melhor experiência de jogo, serão implementados sistemas de configurações e personalização de controlos (keybinds).
Os dados do jogador, como progresso e configurações, serão guardados em ficheiros JSON.
